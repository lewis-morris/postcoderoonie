GET postcoderoonie.co.uk/postcode

This endpoint is used to search for salary with average household salary between two values. Will return a list of location objects. Some postcodes have NULL values for salary and these will be omitted. Due to the possible size of results, the request will be paginated.

GET Parameters

start_sal=[float]
start_sal=[float]
Starting salary value i.e 20000
REQUIRED
end_sal=[float]
end_sal=[float]
Ending salary value i.e 25000
REQUIRED
postcode=[string]
postcode=[string]
Postcode lookup value
OPTIONAL

Extra Information

Results are always sorted in ascending order

Example Usage

salary with start & end value
https://postcoderoonie.co.uk/api/salary?start_sal=10000&end_sal=15000
salary with start, end value & postcode filter
https://postcoderoonie.co.uk/api/salary?start_sal=10000&end_sal=15000&postcode=en1

Example Response


GET https://postcoderoonie.co.uk/api/salary?start_sal=50000&end_sal=55000&postcode=en1

Example Empty Response


GET https://postcoderoonie.co.uk/api/salary?start_sal=999999&end_sal=999999
