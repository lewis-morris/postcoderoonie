Getting started

Postcoderoonie is a API for searching and validating UK postcode data, its free and easy to use and accessible for all.

Authentication

Requests to the Postcoderoonie API do not require authentication, however, requests will be limited by IP to 3 request a second / 500 per hour.

HTTP method

Every call to the Postcoderoonie API should be a GET request.

Data source

This amazing wealth of information has been acquired from https://www.doogal.co.uk/, if you want to download the data for your own use, or find some of the other amazing postcode related information, head over to Chris' site.

I'm sure many of us owe Chris a debt of gratitude. A for effort Chris.

Data coverage

Due to source limitations, some data may be incorrect or missing. This resource cannot be relied upon and must be treated as inaccurate (although we believe it to be good). This service is provided "as is" and we cannot accept any liability for any misinformation provided by the API.

Response times

Due to the large amount of records in the database & the cheap server this is running on, some calls, especially larger lists of results may take some time to return.
