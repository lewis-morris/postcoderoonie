GET postcoderoonie.co.uk/postcodes

This endpoint is used to lookup partial matches of postcodes and will return a list of location objects. Due to the possible size of results, the request will be paginated.

GET Parameters

postcode= [string]
postcode= [string]
Postcode partial lookup value
REQUIRED (min-length=3)

Example Usage

partial postcode lookup
https://postcoderoonie.co.uk/api/postcodes?postcode=PE14
search postcode, with 5 results and page 10
https://postcoderoonie.co.uk/api/postcode?postcode=PE14&results=5&page=10
