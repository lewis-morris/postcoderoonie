GET postcoderoonie.co.uk/postcode

This endpoint is used to search for postcodes with exact matches and will return a single location object or an array if multiple searches are conducted.

GET Parameters

postcode=[string]
postcode=[string]
Postcode lookup value (can produce multiple results by adding the | pipe between search terms
REQUIRED (min-length=4)
{% include "params/extra.html" %}

Example Usage

single postcode with no space
https://postcoderoonie.co.uk/api/postcode?postcode=PE147AP
single postcode with space (switch it for a + )
https://postcoderoonie.co.uk/api/postcode?postcode=PE14+7AP
multiple postcodes
https://postcoderoonie.co.uk/api/postcode?postcode=PE147AP|CT48ER|SA181BE

Example Response


GET https://postcoderoonie.co.uk/api/postcode?postcode=CT48ER

Example Empty Response


GET https://postcoderoonie.co.uk/api/postcode?postcode=err
