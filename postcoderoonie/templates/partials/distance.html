GET postcoderoonie.co.uk/
distance

This endpoint is used to search calculate distance between two postcodes and will return (on valid lookup) two location objects and the distance in specified unit of measurement.
{% with id="distance" %} {% include "params/msreview.html" %} {% endwith %}

GET Parameters

postcode_one=[string]
postcode_one=[string]
First Postcode lookup value (exact match)
REQUIRED (min-length=5)
postcode_two=[string]
postcode_two=[string]
Second Postcode lookup value (exact match)
REQUIRED (min-length=5)
unit=[string]
unit=[string]
Unit of measurement - can be either m (for miles) or km (for kilometres)
Optional (default: m)

Example Usage

distance between two postcodes in miles
https://postcoderoonie.co.uk/api/distance?postcode_one=PE147AP&postcode_two=EN118RX
distance between two postcodes in miles
https://postcoderoonie.co.uk/api/distance?postcode_one=PE147AP&postcode_two=EN118RX&unit=km

Example Response


GET https://postcoderoonie.co.uk/api/distance?postcode_one=PE147AP&postcode_two=EN118RX

Example Invalid Response
